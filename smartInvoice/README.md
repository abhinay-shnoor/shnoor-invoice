# Smart Invoice & Expense Management System — Landing Page

A modern, responsive, and enterprise-grade SaaS landing page built to showcase our company products and business solutions.

This platform presents:

- Smart Invoice & Expense Management System
- HRMS Platform
- Workspace Management Platform

Designed with a clean and professional UI inspired by modern SaaS platforms like **Stripe**, **Linear**, **Notion**, and **Razorpay**.

---

# 🚀 Features

- Modern SaaS-style landing page
- Fully responsive design
- Dark / Light mode support
- Smooth animations using Framer Motion
- Sticky responsive navbar
- Product showcase section
- Workflow visualization
- Dashboard preview UI
- Testimonials section
- Professional footer
- Mobile-friendly layout

---

# 🛠️ Tech Stack

## Frontend

- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide React Icons

# ⚡ Installation

## 1. Clone Repository

```bash
git clone https://github.com/yourusername/smart-invoice-system.git
```

## 2. Navigate to Project

```bash
cd smart-invoice-system
```

## 3. Install Dependencies

```bash
npm install
```

## 4. Start Development Server

```bash
npm run dev
```
